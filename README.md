## 🚀 주요 기능

가천대학교 및 인천대학교 학생들의 편리한 학습 관리를 위해 강력한 기능들을 제공합니다.

1. **녹화강의 완벽 지원**
   - 녹화강의의 **마감 기한**과 **학습 진행도(%)**를 실시간으로 확인하고 바로 시청 페이지로 이동할 수 있어요.
2. **퀴즈 현황 추적**
   - **형성평가 및 퀴즈**의 제출 여부와 마감 기한도 대시보드에서 함께 관리할 수 있어요.
3. **실시간 마감 카운트다운**
   - 서버 갱신 없이도 **분 단위로 줄어드는 남은 시간**을 실시간으로 확인할 수 있어요.
4. **강력한 멀티 필터링**
   - 상태(진행 중, 마감 임박 등), 카테고리, 과목별로 원하는 정보만 골라볼 수 있어요.
5. **어디서나 접근 가능**
   - 사이버캠퍼스뿐만 아니라 **모든 웹사이트**에서 대시보드를 열어 마감 기한을 체크할 수 있어요.
6. **최적화된 UI/UX**
   - 한국어 과목명 자동 정제, 부드러운 애니메이션, 가독성이 개선된 로딩 진행 바를 제공해요.

---

## 🤔 프로젝트 소개

사이버캠퍼스의 파편화된 정보를 한 곳에서 모아보고 싶다는 니즈를 충족하기 위해, 기존의 도구들을 계승하고 현대적인 환경(녹화강의, 퀴즈 등)에 맞춰 완전히 새롭게 리빌딩된 프로젝트입니다.

---

## 📝 이용 방법

1. **설치**: 확장 프로그램을 브라우저에 추가합니다.
2. **접근**: 사이버캠퍼스 로그인 후, 우측 하단 아이콘을 클릭하거나 단축키(`Ctrl + /` 또는 `Cmd + /`)를 누르세요.
3. **동기화**: 갱신 버튼을 눌러 최신 과제 정보를 가져옵니다.

## ℹ️ 개인정보 보호

모든 데이터는 브라우저의 **로컬 스토리지**에만 저장되며, 외부 서버로 절대 전송되지 않습니다.
